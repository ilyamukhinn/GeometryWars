## Краткое описание

Данный проект посвящен продолжению изучения (после [GeometricBouces][GeometricBouces]) с [SFML][SFML-url] и [ImGui][ImGui-url]

## Зависимости

* Библиотека SFML (2.6.1)

## Сборка и запуск

Вы можете открыть проект в среде разработки Visual Studio (на момент написания программы использовалась VS 2022), в корневой директории располагается файл проекта "GeometryWars.vcxproj". Проект сконфигурирован как для Debug, так и для Release сборки.

После сборки исполняемый *.exe файл будет находится в директории "\*Корневая папка проекта\*/\*Имя проекта/.../bin/\*.exe\*"

## После запуска

На экране в центре будет представлена геометрическая фигура - игрок. Вы можете выполнять следующие действия будучи игроком:
- W - движение вверх
- A - движение влево
- S - движение вниз
- D - движение вправо
- Left mouse click - запуск снаряда (выстрел из оружия) по направлению местонахождения курсора во время нажатия
- Right mouse click - запуск снарядов (выстрел из оружия) вокруг игрока

Во время игры на экране в хаотичном порядке появляются геометрические фигуры (противники). Вам необходимо стрелять в них, чтобы уничтожить и получить игровые очки (см. верхний левый угол игрового окна). При успешном попадании они распадаются на свои уменьшенные копии, которые также являются активными целям (однако, их время жизни ограничено), за них вы сможете получить в два раза больше очков, чем за оригинальную фигуру.

При столкновении со стенками экрана или противником игрок перерождается в центре экрана. При этом, если столкновение произошло с противником, то вы теряете все набранные очки.

Также вы можете увидеть графический интерфейс с двумя вкладками: Systems и Entities.

Во вкладе Systems вы можете управлять следующими системами игры:
- Lifespan: время жизни, если отключить данную систему, то уменьшенные копии противников, а также ваши снаряды не будут исчезать по прошествии времени
- Spawning - система спавна, если отключить её, то противники перестанут 
возраждаться. Также ваш игрок не будет возрожден, если во время отключения данной системы вы погибните
- Movement - система движений, если отключить её, все объекты на экране перестанут менять своё положение
- Collision - система коллизий, если отключить её, все объекты на экране перестанут сталкиваться
- User input - система ввода от пользователя, если отключить её, вы не сможете выполнять какие-либо действия с помощью нажатий на клавиши на клавиатуре и мыши
  
Помимо выше указанных системы, вы можете поставить игру на паузу, если нажмете на клавшиу P на клавиатуре.

Во вкладке Entities вы можете наблюдать за активными объектами: снарядами, противниками, игроков и уменьшенными копиями противников

Также для проекта создан config.txt файл, который расположен в папке configs. Вы можете редактировать его перед запуском, спецификация файла:

- Window W H FL FS: строка определяет параметры окна приложения
  
| Смысл           | Имя в файле | Тип         |
|:----------------|:------------|:-----------:|
| Window width    | W           | int         |
| Window height   | H           | int         |
| Frame limit     | FL          | int         |
| Full screen     | FS          | int (0,1)   |

- Font F S R G B: строка определяет путь до файла шрифтов и соответствующие параметры

| Смысл       | Имя в файле | Тип             |
|:------------|:------------|:---------------:|
| Font file   | F           | std::string     |
| Font size   | S           | int             |
| RGB color   | (R,G,B)     | int, int, int   |

- Player SR CR S FR FG FB OR OG OB OT V: строка определяет параметры игрока

| Смысл             | Имя в файле | Тип           |
|:------------------|:------------|:-------------:|
| Shape radius      | SR          | int           |
| Collision radius  | CR          | int           |
| Speed             | S           | float         |
| Fill color        | (FR,FG,FB)  | int, int, int |
| Outline color     | (OR,OG,OB)  | int, int, int |
| Outline thickness | OT          | int           |
| Shape vertices    | V           | int           |

- Enemy SR CR SMIN SMAX OR OG OB OT VMIN VMAX L SI: строка определяет параметры противника

| Смысл             | Имя в файле | Тип           |
|:------------------|:------------|:-------------:|
| Shape radius      | SR          | int           |
| Collision radius  | CR          | int           |
| Min/Max speed     | (SMIN,SMAX) | float, float  |
| Outline color     | (OR,OG,OB)  | int, int, int |
| Outline thickness | OT          | int           |
| Min/Max vertices  | (VMIN,VMIN) | int, int      |
| Spawn lifespan    | L           | int           |
| Spawn interval    | SP          | int           |
